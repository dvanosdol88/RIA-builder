# AI GenConsult Capabilities

This file lists the current, **implemented** capabilities of AI GenConsult in RIA Builder, plus a few **potential additions** that may be helpful in the future.

## Implemented Capabilities

### Google Drive
- **List files**: browse Drive contents (optionally within a folder)
- **Search files**: find Drive files by name or content
- **Read Google Docs**: pull text from a Drive document by file ID
- **Create Google Docs**: generate new documents with title + content
- **Edit Google Docs**: replace or append content in an existing Doc
- **Create folders**: make new Drive folders (optionally under a parent)
- **List folders**: show folders in Drive (optionally under a parent)
- **Rename files/folders**: change names by file ID
- **Copy files**: duplicate a file into a target folder (optional new name)
- **Move files**: move a file into a target Drive folder

### Project Board (Ideas / Kanban)
- **Create cards**: add new ideas or questions with category + subcategory
- **Update cards**: change title, goal, or stage for an existing card

### Slack
- **Send messages**: post updates to the configured Slack channel

### Web Research
- **Web research**: search the web, summarize results, and provide citations

### Local / Uploaded Documents
- **Read uploaded docs**: read full text for documents listed as available in-app

### Memory & Session Context
- **List conversation summaries**: retrieve past session summaries from memory

### Conversation Inputs
- **Attach files**: images, PDFs, and documents can be attached for context
- **Voice input**: speech-to-text transcription for hands-free prompts




     ## Potential Helpful Additions (Not Implemented Yet)

### Communications
- **Email**: read, draft, send, and file emails
- **Calendar**: create, reschedule, and sync meetings
- **Messaging**: Slack / Teams message drafting and sending

### Client Ops & Compliance
- **CRM**: read/update CRM records (e.g., Wealthbox, Redtail)
- **E-sign**: send, track, and file signed documents
- **ADV / compliance**: structured checklists + change tracking

### Documents & Data
- **Spreadsheet editing**: read/write to Sheets/Excel
- **Template library**: generate documents from approved templates
- **Batch processing**: bulk rename, tag, or move files

### Research & Insights
- **Market data**: pull pricing or benchmark data for reports
